# 🧲 Drag & Drop List (HTML, CSS, JavaScript)

A simple drag-and-drop project where users can move list items from one
container to another using pure JavaScript.

------------------------------------------------------------------------

## 🚀 Features

-   Drag items from left box
-   Drop items into right box
-   Smooth UI interaction
-   Clean and minimal design

------------------------------------------------------------------------

## 🛠️ Tech Used

-   HTML
-   CSS
-   JavaScript

------------------------------------------------------------------------

## 📂 Project Structure

Drag-Drop/ │── index.html\
│── style.css\
│── script.js\
│── drag.png

------------------------------------------------------------------------

## ▶️ How to Run

1.  Download or clone the repository\
2.  Open `index.html` in your browser

------------------------------------------------------------------------

## 🎯 How It Works

-   Each list item is **draggable**
-   When dragging starts, the selected item is stored
-   On drop, the item is appended to the right container
